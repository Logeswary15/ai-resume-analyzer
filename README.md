# AI Resume Analyzer (No-Code Project)

## 📌 Overview
This project uses AI (ChatGPT) to analyze resumes and provide insights like:
- Skill gaps
- Resume improvements
- Career suggestions

## ⚙️ Tools Used
- ChatGPT
- Google Docs

## 🚀 How It Works
1. Upload or paste a resume
2. Use structured prompts in ChatGPT
3. Generate insights:
   - Strengths
   - Weaknesses
   - Missing keywords

## 📊 Sample Output
- Suggested skills to add
- Improved resume summary
- Career recommendations

## 📁 Project Outcome
This project helps improve resumes by making them more aligned with job requirements.
